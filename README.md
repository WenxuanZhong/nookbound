# 归隅 / Nookbound

把不同形状的拼块，一点点放回棋盘里。

在线地址：https://nookbound.pages.dev/

## 关于

归隅（Nookbound）是一个运行在浏览器里的拼图游戏。

从选关开始，挑一块棋盘，观察盘面的留白，再把手边的拼块慢慢放进去。它不追求很强的反馈，也不靠密集的节奏推动你往前走；更像是一块可以安静摆弄的棋盘，留一点时间给观察、试探和反复调整。

游戏目前提供按难度分组的关卡选择，也保留了提示、撤销、重开和旋转这些基本操作。

## 体验

- 支持桌面端与移动端
- 可直接拖动拼块进行摆放
- 支持旋转、提示、返回上一步和重开
- 关卡按难度分组，适合逐步推进

## 技术

这是一个纯静态网页项目，不依赖前端框架。构建时会将发布需要的资源整理到 `dist/`，再作为静态站点发布。

## 关于制作

这个项目主要以 AI 辅助的方式完成，可以算是一次很个人的 vibe coding 练习：一边想，一边做，一边把脑海里那种安静、克制的拼图体验慢慢落下来。
